4th slide
“This slide shows the complete system flow of our ISL recognition application.”

“It starts with the user performing a sign in front of the webcam, which is captured using OpenCV.”
“We then use MediaPipe Hands to extract hand landmarks from each frame, which form our dataset.”
“The dataset is split into training and testing sets, and we apply noise augmentation to improve model accuracy and robustness.”
“The trained model, stored as isl_model.pkl, includes both the scaler and the classifier.”
“On the backend side, our Flask application receives real-time frames through a POST API in base64 format.”
“These frames are processed again using MediaPipe, scaled, and passed to a Random Forest model for prediction.”
“Finally, the predicted sign is displayed, and we also provide a text-to-speech feature so the system can speak the detected sign.”
